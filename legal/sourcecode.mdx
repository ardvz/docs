Dokumen ini menjelaskan **syarat**, **batasan**, **dan hak penggunaan** source code Discord bot yang disediakan oleh HostGate kepada Pengguna. Dengan memperoleh, mengunduh, atau menggunakan produk source code dari HostGate, Pengguna secara otomatis dianggap telah membaca, memahami, dan menyetujui seluruh isi kebijakan ini.

## Definisi
- **HostGate:** Penyedia layanan dan pengembang produk digital, termasuk source code Discord bot.
- **Produk Source Code:** File dan skrip yang membentuk aplikasi Discord bot yang disediakan oleh HostGate dalam bentuk _non-binary_ (kode sumber).
- **Lisensi:** Hak terbatas yang diberikan oleh HostGate kepada Pengguna untuk menggunakan source code sesuai dengan syarat kebijakan ini.
- **Pengguna:** Individu atau entitas yang memperoleh akses legal terhadap produk source code dari HostGate.
- **Penggunaan Pribadi:** Pemakaian untuk kebutuhan internal, personal, non-komersial, dan tidak untuk disebarluaskan atau diperjualbelikan.

## Ruang Lingkup Produk
Produk yang dimaksud dalam kebijakan ini adalah **Source Code Discord Bot** yang mencakup:
- File kode utama (_JavaScript_, _TypeScript_, atau _Python_)
- File konfigurasi dan dependensi
- Dokumentasi teknis yang menyertai
Produk ini disediakan **dalam bentuk digital** dan dilindungi oleh **hak kekayaan intelektual** HostGate.

## Hak dan Lisensi Penggunaan
Dengan memperoleh produk ini secara sah dari HostGate, Pengguna diberikan **lisensi terbatas**, **tidak eksklusif**, **tidak dapat dipindahtangankan**, dan **tidak dapat disublisensikan** untuk:
- Menggunakan source code **hanya untuk keperluan pribadi dan non-komersial**
- Menyesuaikan atau memodifikasi source code untuk penggunaan pribadi sendiri
- Menjalankan bot di server pribadi (bukan layanan publik atau dijual ke pihak lain)
<Warning>**Lisensi ini tidak bersifat kepemilikan penuh.** Pengguna hanya menerima hak pakai terbatas dan tetap tunduk pada pembatasan-pembatasan dalam kebijakan ini.</Warning>

## Larangan dan Pembatasan Penggunaan
Pengguna **dilarang keras** melakukan hal-hal berikut:
- **Menjual kembali**, **memperjualbelikan**, **menyewakan**, atau **mendistribusikan** source code, baik dalam bentuk asli maupun hasil modifikasi
- **Mengunggah ke platform publik** seperti GitHub, GitLab, Replit, dan sejenisnya, kecuali untuk penggunaan pribadi yang bersifat privat (private repository)
- **Mengklaim sebagai pemilik/pembuat utama** dari source code
- **Menghapus atau mengubah lisensi**, **watermark**, atau **keterangan hak cipta** yang disertakan oleh HostGate
- **Menggunakan source code untuk layanan publik**, seperti jasa pembuatan bot untuk klien atau penyewaan bot tanpa izin tertulis
- **Membagikan kepada pihak ketiga**, baik secara gratis maupun berbayar
<Note>Setiap pelanggaran terhadap poin ini akan dianggap sebagai **pelanggaran hak cipta dan lisensi**, dan dapat dikenakan tindakan hukum sesuai peraturan yang berlaku.</Note>

## Perlindungan Hak Kekayaan Intelektual
- Semua elemen dari source code, termasuk struktur, logika, dan dokumentasi adalah milik eksklusif HostGate dan dilindungi oleh **UU Hak Cipta** dan **UU ITE** di Indonesia.
- Pengguna **tidak mendapatkan hak atas kekayaan intelektual**, melainkan hanya hak pakai pribadi.
- HostGate berhak untuk **melacak**, **mengaudit**, atau **menindak pelanggaran lisensi** dengan dukungan alat pelacakan kode (_code fingerprinting_) jika diperlukan.

## Keamanan dan Tanggung Jawab Pengguna
- Pengguna bertanggung jawab atas **keamanan data login**, **token bot**, dan **konfigurasi pribadi** yang digunakan dalam source code.
- HostGate tidak bertanggung jawab atas **kerusakan**, **kehilangan data**, atau **kebocoran** yang terjadi akibat penggunaan source code, baik karena kelalaian pengguna maupun modifikasi yang dilakukan.

## Dukungan dan Pembaruan
- HostGate dapat menyediakan **dukungan teknis terbatas**, sesuai dengan paket atau kesepakatan awal.
- Pembaruan (jika disediakan) akan diberikan hanya kepada Pengguna yang **masih memiliki akses sah** dan tidak melanggar kebijakan.
- HostGate tidak berkewajiban memberikan update atau dukungan jangka panjang kecuali dinyatakan dalam kontrak khusus.

## Pengakhiran Lisensi
Lisensi penggunaan dapat dihentikan secara sepihak oleh HostGate jika:
- Pengguna terbukti **melanggar ketentuan** dalam kebijakan ini
- Terdapat **indikasi distribusi ilegal** atau pembajakan
- Atas permintaan resmi dari pihak berwenang atau karena alasan hukum
Setelah lisensi dihentikan:
- Pengguna **wajib menghapus seluruh salinan source code**, baik di lokal maupun di cloud
- Akses terhadap pembaruan dan dukungan **akan dicabut secara permanen**
